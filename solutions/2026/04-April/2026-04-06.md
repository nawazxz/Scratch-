# Longest Common Prefix

**Topic:** string  
**Language:** python  
**Difficulty:** Easy  

## Solution

```python
def longest_common_prefix(strs):
    prefix = strs[0]
    for s in strs[1:]:
        while not s.startswith(prefix):
            prefix = prefix[:-1]
            if not prefix:
                return ""
    return prefix

print(longest_common_prefix(["flower","flow","flight"]))  # "fl"
```

## Key Insight
Use the first string as candidate; shrink it until every string starts with it.
