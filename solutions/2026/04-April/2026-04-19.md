# Climbing Stairs

**Topic:** dynamic programming  
**Language:** python  
**Difficulty:** Easy  

## Solution

```python
def climb_stairs(n):
    a, b = 1, 1
    for _ in range(n - 1):
        a, b = b, a + b
    return b

# ways(n) = ways(n-1) + ways(n-2)  → Fibonacci!
for i in range(1, 8):
    print(f"stairs={i}: {climb_stairs(i)} ways")
```

## Key Insight
Each step you can come from n-1 or n-2 — identical to Fibonacci. O(n) time, O(1) space.
