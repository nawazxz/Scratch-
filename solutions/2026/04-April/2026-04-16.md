# Reverse Linked List

**Topic:** linked list  
**Language:** python  
**Difficulty:** Easy  

## Solution

```python
class Node:
    def __init__(self, val=0, nxt=None):
        self.val, self.next = val, nxt

def reverse(head):
    prev = None
    cur  = head
    while cur:
        nxt       = cur.next
        cur.next  = prev
        prev, cur = cur, nxt
    return prev  # new head
```

## Key Insight
Three-pointer walk: keep prev, current, and next. Re-wire one node per step.
