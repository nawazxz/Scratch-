# Binary Search

**Topic:** binary search  
**Language:** python  
**Difficulty:** Easy  

## Solution

```python
def search(nums, target):
    lo, hi = 0, len(nums) - 1
    while lo <= hi:
        mid = (lo + hi) // 2
        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            lo = mid + 1
        else:
            hi = mid - 1
    return -1

print(search([-1,0,3,5,9,12], 9))  # 4
```

## Key Insight
Halve the search space each iteration — O(log n) time.
