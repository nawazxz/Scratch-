# Valid Parentheses

**Topic:** stack  
**Language:** python  
**Difficulty:** Easy  

## Solution

```python
def is_valid(s):
    stack = []
    pairs = {')': '(', '}': '{', ']': '['}
    for ch in s:
        if ch in '({[':
            stack.append(ch)
        elif not stack or stack[-1] != pairs[ch]:
            return False
        else:
            stack.pop()
    return len(stack) == 0

print(is_valid("()[]{}"))  # True
```

## Key Insight
Push open brackets, pop & verify on close. If stack is empty at end — valid.
