# Product of Array Except Self

**Topic:** prefix products  
**Language:** python  
**Difficulty:** Medium  

## Solution

```python
def product_except_self(nums):
    n   = len(nums)
    out = [1] * n
    # Left pass
    prefix = 1
    for i in range(n):
        out[i] = prefix
        prefix *= nums[i]
    # Right pass
    suffix = 1
    for i in range(n-1, -1, -1):
        out[i] *= suffix
        suffix *= nums[i]
    return out

print(product_except_self([1,2,3,4]))  # [24,12,8,6]
```

## Key Insight
Two-pass prefix/suffix product — no division, O(n) time, O(1) extra space.
