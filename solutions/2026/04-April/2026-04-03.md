# Merge Intervals

**Topic:** sorting  
**Language:** python  
**Difficulty:** Medium  

## Solution

```python
def merge(intervals):
    intervals.sort(key=lambda x: x[0])
    merged = [intervals[0]]
    for start, end in intervals[1:]:
        if start <= merged[-1][1]:
            merged[-1][1] = max(merged[-1][1], end)
        else:
            merged.append([start, end])
    return merged

print(merge([[1,3],[2,6],[8,10],[15,18]]))
```

## Key Insight
Sort by start time. Extend last interval if overlap, else append new one.
