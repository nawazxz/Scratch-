# Detect Cycle in Linked List

**Topic:** two pointers  
**Language:** python  
**Difficulty:** Easy  

## Solution

```python
class Node:
    def __init__(self, val):
        self.val  = val
        self.next = None

def has_cycle(head):
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
        if slow is fast:
            return True
    return False
```

## Key Insight
Slow moves 1 step, fast moves 2. If they ever meet, there's a cycle — Floyd's algorithm.
