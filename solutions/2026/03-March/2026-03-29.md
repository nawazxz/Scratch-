# Two Sum

**Topic:** hash map  
**Language:** python  
**Difficulty:** Easy  

## Solution

```python
def two_sum(nums, target):
    seen = {}
    for i, n in enumerate(nums):
        complement = target - n
        if complement in seen:
            return [seen[complement], i]
        seen[n] = i
    return []

# Example
print(two_sum([2, 7, 11, 15], 9))  # [0, 1]
```

## Key Insight
Store each number's index in a hash map. For every element, check if its complement already exists — O(n) time.
